# Any public recipe can be added to Weekly Plan or Collections without a Like

Both the Weekly Plan and Collections allow any public Recipe from any User as eligible content — a Like is not required. Previously, the Weekly Plan restricted eligibility to own recipes plus Liked public recipes.

## Considered options

**Liked recipes only (original Weekly Plan behaviour).** A Like acts as the "save" primitive — you must express appreciation before you can organise. Keeps the eligible set small and intentional.

**Any public recipe (adopted).** Like is defined in the glossary as "an expression of appreciation", not a save action. Requiring a Like before adding to a plan or collection conflates two distinct intents. A user might want to plan or collect a recipe they are curious about but do not want to publicly appreciate, or that they own but haven't liked. Forcing a Like to unlock eligibility muddies the Like concept.

## Consequences

The Weekly Plan recipe picker exposes three tabs — "My Recipes", "Liked Recipes", and "All Recipes" — so the Liked Recipes surface remains useful and prominent even though it is no longer a gate.
